
# Anaconda commands

## create environment
```
conda create -n<environment name>
```

## delete environment
```
conda env remove -n<env name>
```

## environment list check
```
conda env list
```

# Virtual env python

## create virtual environment in python
```
python3 -m venv <env name>
```

## activate it
```
<environment name>\Scripts\activate
```

## deactivate it
```
deactivate\
```
